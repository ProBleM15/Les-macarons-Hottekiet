CONSEIL DE DÉGUSTATION

Les macarons Hottekiet se conservent dans votre réfrigérateur et se consomment dans les 7 jours suivant leur achat (macarons sucrés), sinon les macarons salés se conservent dans votre réfrigérateur et se consomment dans les 2 jours.
